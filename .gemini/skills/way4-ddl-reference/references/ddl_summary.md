# Project DDL Summary

This file provides a summary of the tables included in the DDL reference.

## Tables Included
- `CLIENT`: Thông tin khách hàng (Cá nhân/Tổ chức).
- `ACNT_CONTRACT`: Thông tin hợp đồng/tài khoản.
- `ADDRESS_TYPE`: Danh mục loại địa chỉ.
- `AFFILIATION_TYPE`: Danh mục loại liên kết.
- `APPL_PRODUCT`: Danh mục sản phẩm ứng dụng.
- `BRANCH`: Danh mục chi nhánh.
- `CLIENT_TITLE`: Danh mục danh xưng khách hàng.
- `CLIENT_TYPE`: Danh mục loại khách hàng.
- `COUNTRY`: Danh mục quốc gia.
- `CURRENCY`: Danh mục tiền tệ.
- `LANG`: Danh mục ngôn ngữ.
- `MARITAL_STATUS`: Danh mục tình trạng hôn nhân.
- `SALUTATION`: Danh mục lời chào.

## Usage
To view the full DDL of a specific table, search for `CREATE TABLE "INT"."TABLE_NAME"` in [project_ddl.sql](project_ddl.sql).
